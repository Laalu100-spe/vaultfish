🐟 VaultFish
Swim across all your clouds.
VaultFish is an open source multi-cloud file manager that unifies Google Drive, Dropbox, and OneDrive into one intelligent interface. Built with React, TypeScript, Supabase, and Lovable.
Features

Unified dashboard across all connected cloud accounts
Multiple accounts per platform (3 Google Drives, 2 Dropboxes)
Smart duplicate detection and storage optimization
AI-powered file intelligence powered by Claude API
Private gallery with timeline view
Smart Clean — reclaim storage across all clouds
Real-time sync with Supabase

Tech Stack
React · TypeScript · Vite · Supabase · Tailwind CSS · shadcn/ui · Claude API
